# Case 관측 원본 장부

이 문서는 채팅으로 받은 Case 1~3의 관측 자료를 정규화해 보존하는 장부다.
평가 결과를 보고 원본값을 고치지 않는다. 오기 수정이 필요하면 기존 값을
삭제하지 않고 정정 이력을 남긴다.

## 기록 원칙

- 사용자가 보낸 원문을 가능한 한 그대로 보존한다.
- 스톱워치 원본 랩과 사용자가 정리한 구간을 모두 남긴다.
- 캡처 파일명과 캡처 시각을 기록한다.
- 계산하거나 추정한 값은 원본과 분리해 `파생값`이라고 표시한다.
- 누락값은 임의로 채우지 않고 `미확인`으로 둔다.
- 경량·기술 검증의 결과는 이 문서에 섞지 않는다.

## Case 1

### 상태

`관측 대기`

### 이동 전 원본

```text
미입력
```

### 이동 후 원본

```text
미입력
```

### 첨부 캡처

| 구분 | 파일 | 캡처시각 | 확인 상태 |
|---|---|---|---|
| 최초 추천 경로 |  |  | 미입력 |
| 도보 시작 직전 버스정보 |  |  | 미입력 |
| 정류장 도착 직후 버스정보 |  |  | 미입력 |
| 탑승 직전 버스정보 |  |  | 미입력 |

### 정규화된 연결 이벤트

`관측 후 작성`

### 누락·정정·제외 사유

`없음`
